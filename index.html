📡 API Library

A curated library of 120 APIs worth knowing — real free-tier quotas, auth type, country of origin, and copy-paste curl / fetch / Python calls for every one — plus a 1,392-API extended catalog.

API ≠ MCP. An API is a service's raw HTTP interface — you call it directly from code, any language, no AI needed. An MCP server is a connector that wraps APIs so AI agents can use them. This library is the raw layer: pick an API here, call it from your own code — or check the MCP Directory for an agent-ready wrapper.
Free = works end-to-end at $0 · Freemium = free tier exists, real usage needs a paid plan · Paid = paid product/API required · flag = company HQ, 🌐 = global/community
⚠️ Key hygiene: never ship API keys in frontend code or commits — keep them server-side or in env vars. Quotas listed are point-in-time; check the provider's pricing page before you build on one.
